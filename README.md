# Uber Data Analysis Dashboard (Tableau)
## Project Overview
This project explores Uber ride data through an interactive Tableau dashboard, transforming raw trip data into actionable insights on booking patterns, revenue trends, and customer behavior.
The dashboard provides a comprehensive view of ride activity across time, location, and trip characteristics, enabling data-driven decision-making.
## Objectives
	- Analyze ride demand trends over time
	- Identify peak booking periods and high-performing locations
	- Understand trip distance patterns and fare relationships
	- Present key performance indicators (KPIs) in a clear, interactive format
## Dashboard Features
1. Time-Based Analysis
  - Area Chart: Total Trips by Month
    July recorded the highest trips (12,889)
	- Area Chart: Revenue by Month
	  March generated the highest revenue (4.57M)
2. Booking Behavior Insights
	- Bar Chart: Total Bookings by Day
	  Monday had the highest bookings (21,622)
	- Histogram: Total Bookings by Hour
	  Peak booking time: 18:00 hrs (12,390 bookings)
3. Location-Based Performance
	- Bar Chart: Revenue by Pickup Location
	  Barakhamba Road generated the highest revenue (0.34M)
4. Trip Characteristics
	- Bar Chart: Trip Distance Distribution
	- Scatter Plot: Fare vs Distance
	- Highlights correlation between trip distance and fare pricing
## Key Performance Indicators (KPIs)
	•	Total Bookings: 148.77K
	•	Total Revenue: 51.85M
	•	Average Ride Distance: 24.64 km
	•	Average Booking Value: 508.30
## Interactivity
The dashboard includes dynamic filters for:
	- Date
	- Pickup Location
These filters allow users to drill down into specific time periods and locations for deeper insights.
## Tools & Technologies
	- Tableau (Data Visualization)
## Key Insights
	- Ride demand peaks in July, suggesting seasonal or behavioral trends
	- Revenue peaks in March, indicating possible pricing or demand variations
	- Evening hours (around 6 PM) are the busiest for bookings
	- Certain locations consistently outperform others in revenue generation
  Future Improvements
	•	Add predictive analytics for demand forecasting
	•	Incorporate customer segmentation
	•	Integrate geospatial maps for location analysis
